03 Kalite Modul Iliskileri (AS9100)

NCR → CAPA Akisi

NCR acildi (uygunsuzluk tespit)
→ Ciddiyet belirlendi (Minor/Major/Critical)
→ CAPA baslatildi (duzeltici/onleyici)
→ Kok neden analizi yapildi
→ Aksiyon tanimlandi + tamamlandi
→ Etkinlik dogrulamasi
→ NCR kapatildi

FAI → Uretime Izin

Yeni urun / tasarim degisikligi
→ FAI baslat (Ilk Madde Muayenesi)
→ Karakteristik tanimla + olcum yap
→ KABUL → Uretime onay verilir
→ RED → NCR acilir, uretim baslamaz

Muayene → NCR / CoC

Giris/proses/final muayene yapilir
→ FAIL → NCR acilir → CAPA
→ PASS → CoC olusturulur
→ CoC + urun → sevkiyat

SPC → Kontrol Grafigi

Kontrol plani olustur
→ 25+ olcum verisi gir
→ X-bar, R chart otomatik cizilir
→ Cp/Cpk hesaplanir
→ Kontrol disi → Alarm → NCR
→ Kontrol ici → Devam
