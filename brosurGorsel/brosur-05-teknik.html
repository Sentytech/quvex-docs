Guvenlik Katmanlari

Defense-in-depth: 7 katmanli guvenlik mimarisi.

1
Network & Transport
HTTPS/TLS 1.3, HSTS, Nginx reverse proxy, CORS restricted origins
2
Security Headers
CSP, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy
3
Authentication
JWT + Opaque Refresh Token, 12+ char password policy, account lockout (5 attempts / 15min)
4
Authorization
YetkiDenetimi filter, RBAC (Goruntule/Kaydet/Sil), CASL frontend ACL, route guards
5
Input Validation
FluentValidation, parameterized SQL, file upload whitelist (10MB), MIME check, path traversal protection
6
CSRF & Rate Limiting
X-Requested-With / Authorization header required, per-endpoint rate limits, Redis-backed
7
Audit & Monitoring
SecurityAuditService [SECURITY] logs, Correlation IDs (X-Correlation-Id), Sentry error tracking (PII stripped)
