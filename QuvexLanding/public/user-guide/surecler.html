Personel Atama

Vardiya tanımlandıktan sonra personeli ilgili vardiyaya atayın:

Nasıl Yapılır?

  1. IK > Vardiyalar sayfasına gidin
  2. İlgili vardiyayı seçin
  3. Personel Ata butonuna tıklayın
  4. Listeden personel seçin ve kaydedin

Sesli Bildirim Ayarları

Bildirimlerin sesli olarak da duyurulmasını istiyorsanız:

  1. Sağ üst köşedeki zil simgesine tıklayın
  2. Bildirim Ayarları bağlantısını seçin
  3. "Sesli Bildirim" seçeneğini açın
  4. Hangi bildirim türleri için ses istediğinizi işaretleyin
Bilgi
SignalR hub adresi: /hubs/notification. Tarayıcınız WebSocket desteklemiyorsa otomatik olarak long-polling'e geçer.

Excel Import / Export

Toplu veri yükleme ve veri indirme işlemleri Excel dosyaları üzerinden yapılır.

Import (Toplu Veri Yükleme)

Nasıl Yapılır?

  1. İlgili sayfada (ör. Ürünler, Müşteriler) Excel Import butonuna tıklayın
  2. Şablon İndir butonuyla doğru formatta şablon dosyasını indirin
  3. Şablonu doldürün (ilk satır başlık, silinmemeli)
  4. Doldurduğunuz dosyayı yükleyin
  5. Önizleme ekranında verileri kontrol edin
  6. İçe Aktar butonuyla kaydedin

Export (Veri İndirme)

Her liste sayfasında Excel Export butonu bulunur. Mevcut filtreler uygulanmış haliyle veriyi indirir.

Dikkat
Import sırasında zorunlu alanlar boş bırakılamaz. Hatalı satırlar listesi import sonrası gösterilir, başarılı satırlar kaydedilir.

Sprint 11 Yeni Surec Akislari (Nisan 2026)

Sprint 11 ile eklenen 4 yeni is sureci: HACCP, Recall, WPS/WPQR onay ve Kalip bakim dongusu.

HACCP Surec Akisi (Gida Guvenligi)

1. PRP Tanimlama → 2. Tehlike Analizi → 3. CCP Belirleme → 4. Kritik Limit → 5. Izleme → 6. Sapma -> NCR → 7. Dogrulama
AdimKimNe YaparSonuc
1. PRP TanimlamaKalite MuhendisiHijyen, temizlik, personel kontrol proseduru olustururPlan dokumani
2. Tehlike AnaliziHACCP EkibiBiyolojik, kimyasal, fiziksel tehlikeleri listelerRisk matrisi
3. CCP BelirlemeHACCP EkibiKarar agaci ile kritik kontrol noktalarini bulurCCP listesi
4. Kritik LimitKalite MuhendisiHer CCP icin min/max limit tanimlar (orn: ≥75°C)Izleme parametresi
5. IzlemeOperatorBelirli sikligitada olcum alir, sisteme girerCCP kaydi
6. SapmaSistemLimit disi degerde NCR acar, lot karantinaya alir, SignalR+WhatsApp bildirim gonderirDuzeltici faaliyet
7. DogrulamaKalite MuduruAylik HACCP audit, plan gozden gecirmeRevize plan

Recall (Geri Cagirma) Workflow

1. Tetik → 2. Lot Analizi → 3. Musteri Tespiti → 4. Bildirim → 5. Iade Toplama → 6. Imha/Rework → 7. Kapanis Raporu

Tetikleyiciler: Musteri sikayeti, ic NCR, HACCP sapmasi, yasal denetim bulgusu, FSCA (medikal/guvenlik kritik). Sistem lot izlenebilirligi ile hangi musteriye hangi seri numarasinin gittigini otomatik listeler.

WPS / WPQR Onay Sureci (Kaynak)

1. WPS Taslak → 2. Kaynak Muhendisi Onay → 3. Test Kaynagi → 4. NDT + Mekanik Test → 5. WPQR Kaydi → 6. WPS Onayi → 7. Uretim Kullanim

EN ISO 15614: Test kaynagi akreditasyonlu laboratuvarda yaptirilmalidir. Cekme, egme, cizik testleri raporlanir. WPQR belgesi 5 yil gecerli olup ayni parametrelerle yapilacak seri uretim icin dayanak tesk eder.

Kalip Bakim Dongusu

1. Kalip Kayit → 2. Uretim + Shot Sayac → 3. Esik %80 → 4. Preventif Bakim IE → 5. Kontrol + Temizlik → 6. Yeniden Uretime

Omur Yonetimi: Her kalip icin toplam atim omru (orn: 1 milyon shot) tanimlanir. %50, %80 ve %95 esiklerinde farkli uyarilar tetiklenir. %100'e ulasilmadan kalip yenilenmelidir.

Ipucu
Yeni surecleri denemek icin Ayarlar > Sektor Profili'nden ilgili sektor demo verisini yukleyebilirsiniz (Gida, Savunma, Plastik).
